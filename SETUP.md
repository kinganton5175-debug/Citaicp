# 🚀 دليل رفع البوت على Railway

## الخطوة 1 — عمل Bot على Telegram (دقيقتين)

1. افتح تليجرام وابحث عن **@BotFather**
2. ابعتله: `/newbot`
3. اختارله اسم مثلاً: `Cita ICP`
4. اختار username مثلاً: `mi_cita_icp_bot`
5. هيبعتلك **TOKEN** — احتفظ بيه ✅

## الخطوة 2 — معرفة Chat ID بتاعك

1. ابحث عن **@userinfobot** على تليجرام
2. ابعتله `/start`
3. هيبعتلك رقم — ده الـ **ADMIN_ID** بتاعك ✅

## الخطوة 3 — رفع الكود على GitHub

1. روح على **github.com** وسجّل دخول (أو اعمل حساب مجاني)
2. اضغط **New repository**
3. سمّيه `cita-bot` واضغط **Create**
4. ارفع الملفات الأربعة:
   - `bot.py`
   - `requirements.txt`
   - `Procfile`
   - `railway.json`

> **طريقة سهلة للرفع:** اضغط "uploading an existing file" في صفحة الـ repo

## الخطوة 4 — نشر على Railway (مجاناً)

1. روح على **railway.app** وسجّل بحساب GitHub
2. اضغط **New Project** ← **Deploy from GitHub repo**
3. اختار الـ repo بتاعك `cita-bot`
4. بعد ما يكتشف الكود، اضغط على المشروع
5. روح على **Variables** وأضف:

```
BOT_TOKEN  = التوكن اللي جالك من BotFather
ADMIN_ID   = الرقم اللي جالك من userinfobot
```

6. اضغط **Deploy** ✅

## الخطوة 5 — جرّب البوت!

1. افتح البوت بتاعك على تليجرام
2. ابعتله `/start`
3. هتظهرلك أزرار — اضغط **▶️ ابدأ المراقبة**
4. البوت هيبدأ يفحص كل 40-80 ثانية
5. لما يظهر موعد هيبعتلك إشعار فوري 🔔

---

## الأوامر المتاحة في البوت

| الأمر | الوظيفة |
|-------|---------|
| `/start` | القائمة الرئيسية مع الأزرار |
| `/startmon` | ابدأ المراقبة |
| `/stopmon` | وقف المراقبة |
| `/status` | شوف الحالة وعدد الفحوصات |
| `/checknow` | فحص فوري دلوقتي |

- Railway بيدي **500 ساعة مجاناً في الشهر** — كافية لشهر كامل
